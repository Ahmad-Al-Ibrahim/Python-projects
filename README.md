# Python Projects

A collection of beginner Python projects built during my first year of Computer Science at Sultan Qaboos University.

## Projects

### 🎮 Game Menu
A menu-driven program with three mini-games:
- Magic 8-Ball — ask a question, get a random answer
- Coin Toss — flip a coin
- Number Guesser — guess a number between 1 and 10

### ⚡ Electrostatic Force Calculator
Calculates the force between two electric charges using Coulomb's law. Includes input validation and classifies the result as Weak, Moderate, or Very Strong.

### 📉 Asset Depreciation Simulator
Simulates how an asset loses value over time using the declining-balance method. Outputs a formatted year-by-year table.

### 🏪 Branch Sales & Profit Tracker
Tracks product sales across multiple branches using nested loops. Generates a profit summary report with performance flags.
